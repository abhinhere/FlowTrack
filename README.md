# FlowTrack

A minimal, focused task management PWA built for daily flow. Manage your tasks across a Kanban board or list view, track daily progress, and build consistent streaks — all with cloud sync, reliable Web Push notifications, and offline support.

> Built by [@abhinhere](https://github.com/abhinhere)

---

## Features

- **Daily Tasks** — Tasks that appear every day, no deadline needed
- **Weekly Tasks** — Recurring tasks on specific days of the week
- **Deadline Tasks** — One-off tasks with a due date
- **Subtasks** — Break any task into a checklist of subtasks
- **Daily Progress** — Visual progress bar tracking today's completion rate
- **Streaks** — Build momentum by completing all tasks each day
- **Kanban & List Views** — Switch between board and list layouts
- **Search** — Instantly filter tasks by title or description
- **Cloud Sync** — Firebase Realtime Database keeps tasks in sync across devices

## Getting Started

### Prerequisites

- Node.js 18+
- A Firebase project (for cloud sync and Google auth)
- VAPID keys (automatically generated by the backend on first launch)

### Installation

```bash
git clone https://github.com/abhinhere/flowtrack.git
cd flowtrack
npm install
```

### Environment Variables

Create a `.env.local` file in the root:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
NEXT_PUBLIC_FIREBASE_DATABASE_URL=
```

### Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## PWA & Notification Setup

- **Desktop (Chrome/Edge)** — Click the install icon in the URL bar, or use the "Install" button in the app header. Grant notification permissions when prompted.
- **Android (Chrome)** — Tap menu → "Add to Home screen".
- **iOS (Safari)** — Tap Share → "Add to Home Screen". Note: iOS requires installing the PWA to the home screen before Web Push notifications can be enabled.
